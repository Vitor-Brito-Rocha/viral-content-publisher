Privacy Policy

Last updated: April 21, 2026

Viral Content Publisher is a personal automation tool used solely by its owner to publish video content to TikTok via the official TikTok Content Posting API.

1. Information We Collect

The App collects only the OAuth access token and refresh token issued by TikTok upon user authorization. These tokens are stored locally on the owner's device and are never transmitted to any third-party server.

2. How We Use Your Information

The tokens are used exclusively to authenticate API requests to TikTok on behalf of the account owner, specifically to upload video files, set captions and hashtags, and check publication status.

3. Data Sharing

The App does not share any data with third parties. No analytics, tracking, or advertising systems are used. All data remains on the owner's local device.

4. Data Retention

OAuth tokens are stored in a local file (tiktok_token.json) on the owner's device. The owner can delete this file at any time to revoke the App's access. Access can also be revoked through TikTok's account settings.

5. Security

All communication with TikTok's API is performed over HTTPS. Tokens are stored locally and are never uploaded to any external server or cloud service.
